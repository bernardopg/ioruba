# Legacy Archive

This directory keeps retired parts of the repository that are still useful for migration audits, historical context, or parity checks.

- `haskell-runtime`: archived Haskell runtime, configs, GTK/TUI experiments, old firmware sketches, and release packaging
- `arduino-audio-controller`: archived Python/GTK prototype
- `github-automation`: archived Pages, metadata-sync, and Release Please tooling
- `audits`: older audit notes kept for reference

The active product path now lives in `apps/desktop`, `packages/shared`, and `firmware/arduino/ioruba-controller`.
